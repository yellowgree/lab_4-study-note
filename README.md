##### 1) pwd
  shows the current path in a hierarchical directory
  
#### 2) cd and ls
 * **cd** : change directory  
 * **ls** : list files and directories   
         arguments:   
              (/        ---> root  
              .         ---> current directory  
              ..        ---> upper-level directory  
              ~         ---> home of current user  
              /[name]   ---> absolute path  
              ./[name]  ---> relative path  
              ../[name] ---> relative path  
              )
